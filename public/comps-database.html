Supports .csv and .xlsx files
Supported formats: .csv and .xlsx (Excel)

Smart column mapping: The app auto-detects common column names like Sale Price, Rent PSF, Cap Rate, Tenant, City, State, etc. For Excel files with separate Sale/Lease sheets, the type is auto-detected from the sheet name.

Automatic geocoding: No lat/lng columns needed. The app geocodes each address automatically using the address, city, state, and zip code from your file. For best results, include a Zip column.

Google Maps Geocoding API Key
Your API key is stored locally in your browser only and never sent anywhere except directly to Google. Get a key at console.cloud.google.com (enable Geocoding API). You get 10,000 free geocoding requests per month.
Data Backup & Transfer
Export all your comp data to a file, then import it into a new version of this app to transfer your work without re-uploading everything. Useful when switching to an updated version.
